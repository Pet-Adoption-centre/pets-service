# Pets Service

Handles CRUD operations for pets.

## API Endpoints

- `/api/getPet` - getPet
- `/api/createPet` - createPet
- `/api/updatePet` - updatePet
- `/api/deletePet` - deletePet

## Getting Started

```bash
npm install
npm run dev
```
